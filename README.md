# 🛒 TradingChango
**Tradeá los Precios, Maximizá el Carrito.**

TradingChango es una plataforma independiente de monitoreo y comparación de precios de supermercados en Argentina. Analizamos tendencias diarias para que los usuarios tomen decisiones financieras inteligentes en sus compras cotidianas.

## 🚀 Características
- 📈 **Gráficos de Tendencias:** Visualización estilo trading de la evolución de precios.
- 🛒 **Chango Inteligente:** Comparativa en tiempo real de cestas de compra en múltiples cadenas.
- 📱 **PWA Ready:** Instalable en dispositivos móviles como una App nativa.
- 🔒 **Security First:** Infraestructura protegida mediante Cloudflare Workers y Row Level Security (Supabase).

## 🛠️ Stack Tecnológico
- **Frontend:** HTML5, CSS3 (Variables dinámicas), JavaScript Vanilla.
- **Base de Datos:** [Supabase](https://supabase.com/) (PostgreSQL).
- **Seguridad:** Cloudflare Workers (Proxy API) & WAF.
- **Automatización:** Python
- **Deployment:** GitHub Pages.
